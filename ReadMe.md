# 🎬 PVR Cinema Ticket Reservation System

A **console-based Python application** that simulates a cinema ticket booking system using **Object-Oriented Programming (OOP)** concepts.

---

## 📌 Project Overview

This project represents a simple ticket reservation system for a cinema (PVR).
Users can register, log in, view seating arrangements, reserve seats, and confirm bookings.

It demonstrates practical use of:

* Classes and Objects
* Data structures (lists, dictionaries)
* User interaction via terminal
* Basic validation and logic handling

---

## 🚀 Features

* 👤 User Registration & Login
* 🪑 Dynamic Seating Map Display
* 🎟️ Seat Reservation System
* 💰 Price Calculation per Seat
* ✅ Booking Confirmation with Total Cost
* 🔁 Reset system for new bookings

## 🎮 How It Works

1. Register as a new user
2. Log in with credentials
3. View available seats
4. Select row and column to book
5. Confirm reservation
6. System calculates total cost

---

## 📊 Sample Seating Format

```
   C0 C1 C2 C3 C4
R0 A  A  A  A  A
R1 A  A  A  A  A
R2 A  A  A  A  A
```

* `A` = Available
* `NA` = Not Available (Booked)
